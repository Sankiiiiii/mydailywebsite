About This Website

This website is created as part of the Web Programming Lab EL for the course curriculum.

It is built using basic HTML and CSS and functions as a simple daily productivity tool. It includes pages for planning, task management, note-taking, and general information.

  • 📝 Planner: Schedule and organize your daily tasks
  • 📋 Tasks: View your tasks in a structured table
  • 📌 Notes: Jot down quick notes and reminders
  • ℹ️ About: About the website and Project

This project demonstrates our understanding of HTML , forms, tables, CSS styling.
